Problem Statement & Objective

Financial markets exhibit high sensitivity to unstructured information such as news, earnings announcements, and macro events. Traditional models fail to incorporate semantic signals effectively.

This project addresses the challenge of predicting short-term stock movement (T+3) by integrating structured technical indicators with unstructured news sentiment using machine learning.

Methodology Overview

Data Pipeline: Integrated multi-source financial data pipeline combining Economic Times & CNBC news scraping (keyword-filtered, MD5 deduplicated ~1200 articles) with Yahoo Finance OHLCV data (2023–present) aligned at daily frequency
Feature Engineering: Derived predictive signals including returns, trend strength, momentum (3D/5D), volatility (5D/10D), volume spike, RSI, MACD, Bollinger width, and aggregated sentiment polarity features
Leakage Prevention: Strict temporal alignment using lagged features (t-1) ensuring models only learn from past data and simulate real-world forward prediction
Models: Supervised learning using Decision Tree, Random Forest, SVM, and Naive Bayes for classification, along with Linear and Ridge Regression for return magnitude prediction
Target: Short-term forecasting of stock behavior including directional movement (3-day UP/DOWN classification) and return magnitude estimation for risk-adjusted decision support
